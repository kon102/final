부트스트랩은 어떻게?

두가지형태로 연동을 할수가 있는데,
css 파일 자체를 다운받아서 link rel로 위치 잡아서 연동해도 되고 CDN 방식으로 연동해도 된다.(※)

Bootstrap 이 연동됬다면, 미리 설계해둔 css 클래스들을 이용이 가능해진다.
그중에 먼저 살펴봐야될 class 들이 layout 을 잡을때 사용하라고 만들어둔 Grid System 에 관련된 class 들이다.
